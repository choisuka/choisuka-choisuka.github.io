# ❤️ 내사랑

부모가 아이에게 화내는 순간을 감지해 감정을 제어할 수 있도록 돕는 앱.
마이크가 목소리를 감지하면 아이 사진과 진정 음향으로 감정을 부드럽게 낮춰줍니다.

**🌐 바로 가기:** https://choisuka.github.io/naesarang

---

## 주요 기능
- 🎙️ 마이크 볼륨 자동 감지
- 📸 아이 사진 표시
- 🎵 4가지 진정 음향 (멜로디·빗소리·자장가·외부링크)
- 🫁 4-4-4 박스 호흡 가이드
- ⏱️ 3분 타이머
- 📊 감지 기록 & 시간대 차트
